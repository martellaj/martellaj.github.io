mls-preview. Who's up next?

As an avid Seattle Sounders and MLS fan, I felt compelled to make something around that. I started out with a very simple Node.js command line tool that let's you check upcoming matches for your favorite teams. It's not elegant, but it's mine.

Install via npm (npm install -g mls-preview).
